## Quick Installation

```bash
# Clone the repository
git clone https://github.com/Baby-root/Baby-Brain.git

# Navigate and make scripts executable
cd Baby-Brain && chmod +x scripts/*.sh

# Launch automation
./scripts/automation.sh
```
